Tally Setup
Configure Tally Prime for integration with Sathi-Connect.
Step 1 — Enable Tally XML Server
  1. Open Tally Prime
  2. Press F12 → Product & Features
  3. Select TDL Management
  4. Set Load TDL files on Startup = YES
  5. Accept and Restart Tally Prime
Step 2 — Download & Load TDL File

Download the TDL file generated with your current configuration and load it into Tally Prime.

Step 3 — Test Connection

Verify that Sathi-Connect can reach Tally Prime on the configured URL.
